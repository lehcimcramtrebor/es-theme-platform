Include this "_support" folder at root of your theme to add developper paypal support.


Then add this line in your theme.xml:


	<include>./_support/support.xml</include>



Like this example:

<theme>
<formatVersion>7</formatVersion>
<include>./_support/support.xml</include>


This will automatically determine if user use Retrobat or Batocera.
You can add your creator QRCODE named creator.png in the folder.
